# Color-WordPress
WordPress资讯自媒体博客微信小程序，前端引入colorUI进行美化。

目前还未优化完，尽量在6月之前优化完并开源，喜欢的话可以点个star。
此小程序是使用WordPress小程序开源版二开而来，引入了colorUI组件。

前期会免费赠送给一部分用户进行测试，优化完善后会开源到github,有需要的可以先点个star。
